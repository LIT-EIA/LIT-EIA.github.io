The deadline to complete the ETC is March 31st, for most employees.

 

The completion deadline for employees who joined the department after April 1st is one year from the date they were assigned the curriculum (see Individual Deadlines in Saba).

 

Please note: These employees are excluded from their team’s ETC completion objectives.

 

Follow these instructions to view an accurate picture of your team’s completion data:

  1. Select the Main menu button, which appears as three lines in the top left hand corner of the page.

  2. Select the My Team item to display the “Direct Team” page. Select the Expand icon, which appears as four (4) tiles in the top left hand corner of the Direct Team page.

  3. Select Manager Dashboards. This opens the Manager Dashboards page. The default report shown is the Summary Report for ESDC’s Essential Training Curriculum, and contains four (4) additional reports.

  4. Select either the “In Progress” box or the “Completed” box of the desired report, which will bring you to the full report.

  5. Select the graphic of your choice, which will bring you to your desired report.

  6. To export your data in Excel, select the download button (Export as a PDF…) appearing as a downward arrow, which is located in the top right hand corner of the page.

For security reasons, please do not share links received from Saba, or copied from your Saba account. If you wish to share a link, please follow these steps:
For a course or class:

  1. Log into your Saba account.
  2. From the Main menu collapse, select the Admin item.
  3. Select the Learning Admin tile.
  4. Select the Manage Classes hyperlink.
  5. Search for your class.
  6. Open Class Details page by selecting the course name.
  7. In the Main tab’s Class Details section, copy the deeplink URL based on your needs. Each option lists two URLs; be sure to copy the shortest link (the second). Recommended options:
  8. Course Deeplink URL: will open the learner’s view of a course.
  9. Class Deeplink URL: will open the learner’s view of a specific class.

Other options (may result in unwanted registrations or no-shows):


  • Registration Deeplink URL: will open the learner’s view of a specific class, AND automatically register the person opening the link for this class.
  • Register and Launch Deeplink URL (only for Web based classes): will automatically register the person opening this link and launch its first activity that is not evaluated.

For a curriculum or certification:

You can also share a deeplink to a curriculum or certification page for learners to register for all classes. This is especially useful when a course can only be accessed through a curriculum or certification, allowing learners to register for classes directly from that page.

  1. From the Learning Admin menu, select the Manage Learning Catalog hyperlink.
  2. Search for the curriculum or certification.
  3. Open Curriculum or certification’s details page.
  4. Copy the Deeplink URL.
