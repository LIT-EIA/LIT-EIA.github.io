Be suspicious of unexpected email attachments. The easiest way to identify whether a file is dangerous is by its file extension, which tells you what type of file it is.

Some dangerous file extensions include:

  • .msi,
  • .bat,
  • .com,
  • .cmd,
  • .hta,
  • .scr,
  • .pif,
  • .reg,
  • .js,
  • .vbs,
  • .wsf,
  • .cpl,
  • .jar,
  • .exe,
  • .docm,
  • .xlsm,
  • .pptm,
  • .zip,
  • .rar,
  • .7z
